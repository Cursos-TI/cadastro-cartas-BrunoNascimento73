# SUPER TRUNFO
Projeto de desenvolvimento do jogo Super Trunfo, como parte da disciplina de Super Trunfo em C. Versão digital do jogo de cartas "Super Trunfo".

## Descrição
O objetivo é aprender a implementação e estrutura da linguagem C. O jogo permite que os jogadores comparem dois atributos de duas cartas, representados por cidades do Brasil, para decidir quem é o vencedor.

## Feito em C

## Como executar
Abra o terminal e navegue até o diretório onde seu arquivo está salvo. Use o gcc para compilar o código:

gcc -o Nome_do_Programa.c

Isso cria um arquivo executável chamado Nome_do_Programa.

## Executar:
Execute o programa no terminal digitando:

./Nome_do_Programa
